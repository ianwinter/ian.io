To say Hello, World! in ColdFusion, using CFM:

``` cfm
<cfoutput>
Hello, World! The time is dateFormat("2017-09-11", "full")#.
</cfoutput>
```

This will output:

``` html
Hello, World! The time is Monday, September 11, 2017.
```
